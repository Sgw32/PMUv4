#Сборка
##Разеъемы для управления ШИМами
*	Cooler   - ШИМ регулятора импеллера охлаждения
*	Starter  - ШИМ регулятора стартера
*	Injector - ШИМ машинки заслонки газа

Квадратный разъем обозначет общую точку. 

##Разеъемы питания
*	SWITCH  - выключатель
*	Servos_12V - питание 6 Вольт; Правая колодка - общая точка
*	LED - инжикация
