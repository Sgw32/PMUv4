# Сборка
## Разеъемы для управления ШИМами
*	Cooler   - ШИМ регулятора импеллера охлаждения
*	Starter  - ШИМ регулятора стартера
*	Injector - ШИМ машинки заслонки газа

Квадратный разъем обозначет общую точку. 

## Разеъемы питания
*	SWITCH  - выключатель
*	Servos_12V - питание 6 Вольт; Правая колодка - общая точка
*	LED - индикация питания борта
*	BAT;BAT1 (+-) - разъемы подключения АКБ
*	ChgIN (+-) - разъем подключения зарядки АКБ
*	Starter (+-) - разъем питания регулятора стартера
*	Cooler (+-) - разъем питания регулятора охлаждения
*	X1..X8 (+-) - разъем питания нагрузки
*	A;B;C - разъем фаз обмоток генератора
